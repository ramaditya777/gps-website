# Gill Power Solutions - React Landing Page

A modern, responsive single-page website built with **React** and **Tailwind CSS** to showcase and advertise Gill Power Solutions’ electrical and EV services. This website highlights your services, government approval, and makes it easy for clients to contact you directly.  

---

## Features

- **Single-page layout** with smooth scrolling navigation
- **Responsive design** for mobile, tablet, and desktop
- **Hero section** with bold tagline and CTA button
- **Services section** in interactive box format
- **About section** with government approval badge
- **Contact section** with form, phone, email, and WhatsApp links
- **Footer** with branding and copyright
- **Optional extras** (can be added):
  - WhatsApp & Call buttons
  - Gallery of work
  - Smooth animations
  - Dark/Light mode toggle
  - EmailJS integration for direct messages
  - Google Maps integration
  - SEO setup  

---

## Tech Stack

- **React** – Component-based frontend framework
- **Tailwind CSS** – Utility-first CSS framework for rapid styling
- **Optional:** Framer Motion for animations, React Icons, EmailJS
